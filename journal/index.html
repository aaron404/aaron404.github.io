2024-06-21 - Blog is live! 🔧

I have been wanting to try blogging for several years now and finally took the time to do some research on the subject. For now I’ve settled on a markdown-based static site generator called Zola, featuring the anemone theme. I plan to use this blog to document some of my hobby projects. Some of my interests you can expect to hear about are programming, computer graphics, and maybe some music. I’m kicking things off with a blog series about automating Dungeons and Diagrams, a logic puzzle minigame present in the game Last Call BBS.

Built with Zola using anemone theme.
